# 猎鸭小游戏 (Duck Hunting Game)

一款基于 Cocos Creator 开发的经典猎鸭休闲射击游戏，操作简单，趣味十足！

## 📖 游戏介绍
这是复刻"猎鸭"玩法的小游戏，玩家通过瞄准、射击屏幕中出现的鸭子获得分数，随着游戏进行，需要达到的分数会越来越高！
